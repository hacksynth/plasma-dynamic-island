# plasma-dynamic-island

A KDE Plasma 6 panel plasmoid: a morphing capsule anchored beneath the top
panel that will surface notifications, media, OSDs, progress jobs, and timers.
Phase 1 ships only the idle-state skeleton.

## Commands

```bash
kpackagetool6 -t Plasma/Applet --install .
kpackagetool6 -t Plasma/Applet --upgrade .
kpackagetool6 -t Plasma/Applet --remove org.kde.plasma.dynamicisland
```
