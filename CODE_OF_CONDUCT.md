# Code of Conduct

This project is a small, technical open source repository. Keep discussion
focused, respectful, and useful to people trying to build, review, and use
the plasmoid.

## Expected behavior

- Be direct, but stay professional.
- Assume good intent unless there is clear evidence otherwise.
- Prefer actionable technical feedback over personal criticism.
- Keep bug reports, reviews, and design discussions grounded in observed
  behavior, reproducible steps, or concrete tradeoffs.
- Respect the time of maintainers and contributors by doing basic due
  diligence before opening issues or pull requests.

## Unacceptable behavior

- Harassment, threats, discrimination, or abusive language.
- Personal attacks, dogpiling, or repeated bad-faith arguments.
- Publishing private, security-sensitive, or personally identifying
  information without consent.
- Spam, trolling, or intentionally derailing technical discussion.

## Scope

This code of conduct applies to repository issues, pull requests, reviews,
discussions, and other project spaces managed for this repository.

## Reporting

Report conduct problems through a private channel on the repository host when
possible. If no private reporting path is available, open a minimal issue that
requests maintainer contact without publishing sensitive details.

## Enforcement

The maintainer may edit, hide, lock, or remove content that violates this code
of conduct, and may temporarily or permanently restrict participation for
repeat or severe violations.
